{/* Section 02 — Quick Start */}
<div class="home-section">
  <div class="section-eyebrow">02 — Get started</div>
  <h2 class="section-title">Up in 60 seconds</h2>
  <p class="section-subtitle">Copy agents into your tool of choice and start delegating immediately.</p>

```bash
# Claude Code — copy agents to your local directory
git clone https://github.com/wrichman/agency-agents
cp -r agency-agents/* ~/.claude/agents/
```

```bash
# All other tools — auto-detect and install
cd agency-agents
./scripts/convert.sh   # generate tool-specific formats
./scripts/install.sh   # interactive installer
```
