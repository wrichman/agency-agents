## Quick Start

Copy agents to your Claude Code directory and start using them immediately:

```bash
# Clone and install
git clone https://github.com/wrichman/agency-agents
cp -r agency-agents/* ~/.claude/agents/
```

```bash
# For Cursor, Windsurf, Aider, Gemini CLI, or OpenCode
./scripts/convert.sh   # generates tool-specific formats
./scripts/install.sh   # interactive installer (auto-detects your tools)
```

## Supported Tools

| Tool | How Agents Load |
|------|----------------|
| [Claude Code](https://claude.ai/code) | Native `.md` → `~/.claude/agents/` |
| [GitHub Copilot](https://github.com/copilot) | Native `.md` → `~/.github/agents/` |
| [Cursor](https://cursor.sh) | `.mdc` rules → `.cursor/rules/` |
| [Windsurf](https://codeium.com/windsurf) | `.windsurfrules` |
| [Aider](https://aider.chat) | `CONVENTIONS.md` |
| [Gemini CLI](https://github.com/google-gemini/gemini-cli) | Extension + `SKILL.md` files |
| [OpenCode](https://opencode.ai) | `.md` agents → `.opencode/agents/` |
| [Antigravity](https://github.com/google-gemini/antigravity) | `SKILL.md` per agent |
| [OpenClaw](https://github.com/openclaw/openclaw) | `SOUL.md` + `AGENTS.md` + `IDENTITY.md` |

## Contributing

1. Fork the repo and create an agent file in the appropriate category directory
2. Follow the template: frontmatter (`name`, `description`, `color`, `emoji`) + Identity + Core Mission + Deliverables
3. Run `./scripts/lint-agents.sh` to validate
4. Submit a PR

See [CONTRIBUTING.md](https://github.com/wrichman/agency-agents/blob/main/CONTRIBUTING.md) for full details.
